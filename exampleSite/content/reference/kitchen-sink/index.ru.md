## Код {#code}

Go подсвечивается сервером (Chroma в цветах палитры):

```go
type Point struct {
	X, Y int64
}
func dist(a, b Point) int64 { return (a.X-b.X)*(a.X-b.X) }
```

С подписью файла и подсветкой строки (`{label=… hl_lines=[2]}`):

```go {label="runtime/malloc.go (упрощено)" hl_lines=[2]}
func mallocgc(size uintptr) unsafe.Pointer {
	c := getMCache()          // горячий путь — без блокировок
	return c.alloc(size)
}
```

Не-Go — плоский `.nohl` (fence `text` или без языка):

```text
$ go build ./...
ok  taiga/internals  0.312s
```

## Термины {#terms}

Наведи на подчёркнутое слово — определение раскроется карточкой; клик её
закрепит, чтобы можно было спокойно читать и уйти по ссылке. Без JavaScript
все определения лежат списком в конце страницы, и каждое слово ведёт к своему.

Простейшая форма — слово и определение, тип по умолчанию:
у каждого P есть свой {{< term "mcache" >}}
Пер-P кэш мелких объектов. Аллокация из него идёт **без единого атомика** —
в этом весь смысл: быстрый путь не трогает разделяемое состояние.
{{< /term >}}, и попадание в него — самый дешёвый путь аллокации.

С ярлыком и цветом типа, и с ссылкой в подвале карточки:
компилятор вставляет {{< term word="барьер записи" title="Барьер записи" kind="ловушка" color="copper" href="/inside/anatomy/palette-is-a-file/" more="Разбор: барьеры и GC" >}}
Кусок кода, который рантайм вставляет **перед каждой записью указателя** в кучу,
пока GC работает конкурентно.

```go
// упрощённо: runtime/mbarrier.go
func writePointer(slot *unsafe.Pointer, ptr unsafe.Pointer) {
	shade(*slot)        // старое значение — в серые
	*slot = ptr
}
```

Отсюда правило: «указатель, записанный при работающем GC, не может быть потерян».
{{< /term >}} — и именно поэтому у записи указателя цена выше, чем у записи `int`.

Слово в тексте склоняется, а шапка карточки — нет; для этого `title=`:
данные лежат в {{< term word="кэшах ядра" title="Кэш процессора" kind="железо" color="green" >}}
Три уровня — L1, L2, L3. Чем ближе к ядру, тем меньше и быстрее:

- **L1** — десятки килобайт, ~4 такта
- **L2** — сотни килобайт, ~12 тактов
- **L3** — мегабайты, общий на сокет, ~40 тактов

Промах мимо L3 стоит поход в память — это сотни тактов, и именно его
прячет за собой слово «локальность».
{{< /term >}}, а не в оперативке.
