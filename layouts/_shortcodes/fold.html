{{- /* Fold — a minimal inline disclosure. A quiet one-line summary that expands into a panel of markdown, with no border or background, so it reads as part of the prose (not a boxed widget). General-purpose; a "there is also a video version" note is just one use of it. {{< fold icon="video" title="There is a [video version](https://…)" >}} …markdown body… {{< /fold >}} Params: icon= leading glyph, one of the keys in $icons below — video · info · tip · book · code · terminal · warning · star · link · note. Default "info"; use "none" to omit it. title= summary line, rendered as INLINE markdown (links work). Required. more= collapsed toggle label. Default "Подробнее". less= expanded toggle label. Default "Свернуть". open= "true" to render already expanded. Default collapsed. Styling + the height-reveal animation live in assets/css/22-fold.css (.fold …); the height animation needs interpolate-size, set on :root in 00-tokens.css. One tiny once-per-page script keeps a link inside the summary from toggling the panel when it is clicked; without JS the only cost is that such a click also toggles — the component still works. */ -}} {{- $icons := dict "video" `` "info" `` "tip" `` "book" `` "code" `` "terminal" `` "warning" `` "star" `` "link" `` "note" `` -}} {{- $name := .Get "icon" | default "info" -}} {{- $title := .Get "title" -}} {{- if not $title }}{{ errorf "fold: missing required param title= (page %s)" .Page.Path }}{{ end -}} {{- if and (ne $name "none") (not (index $icons $name)) }}{{ warnf "fold: unknown icon %q (page %s)" $name .Page.Path }}{{ end -}} {{- $more := .Get "more" | default "Подробнее" -}} {{- $less := .Get "less" | default "Свернуть" -}} {{- $open := eq (.Get "open") "true" -}}
{{- if and (ne $name "none") (index $icons $name) -}} {{- end -}} {{ .Page.RenderString $title }} {{ $more }}{{ $less }}
{{ .Page.RenderString (dict "display" "block") .InnerDeindent }}
