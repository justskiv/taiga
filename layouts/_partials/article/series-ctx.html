{{- /* Series context for the current article, resolved once and shared by kicker, rail-left, series-bridge, feed cards and OG. A series is a SUB-SECTION of a rubric — the guide lives in content//// — so membership is positional: .CurrentSection is the series exactly when the page sits deeper than its root section (.FirstSection, the rubric) — unless that sub-section is a standalone CONTAINER (params.standalone: true in its _index.md), a tidiness folder whose children are ordinary standalone guides. Returns a dict: isSeries bool — true only when the series has >1 listed part term page — the series section page (its landing) label string — short lowercase series name (params.label | lower title) href string — the series landing URL slug string — series slug (= folder name; anchor id on the rubric) pages dicts — series parts from series/parts.html, weight-asc: {title,linkTitle,railTitle,mins,url,announce,weight}; announced parts (draft+announce) carry url:"" so they render as non-clickable titles and count in pos/total pos,total int — 1-based position of this page, and part count A standalone guide — a direct child of the rubric, or a child of a container — returns {isSeries:false}. */ -}} {{- $ctx := dict "isSeries" false -}} {{- $sec := .CurrentSection -}} {{- if and $sec (not ($sec.Eq .FirstSection)) (not $sec.Params.standalone) -}} {{- $pages := partialCached "series/parts.html" $sec $sec.RelPermalink -}} {{- $pos := 0 -}} {{- range $i, $p := $pages }}{{ if eq $p.url $.RelPermalink }}{{ $pos = add $i 1 }}{{ end }}{{ end -}} {{- $ctx = dict "isSeries" (gt (len $pages) 1) "term" $sec "label" ($sec.Params.label | default (lower $sec.Title)) "slug" (partial "series/slug.html" $sec) "href" $sec.RelPermalink "pages" $pages "pos" $pos "total" (len $pages) -}} {{- end -}} {{- return $ctx -}}
